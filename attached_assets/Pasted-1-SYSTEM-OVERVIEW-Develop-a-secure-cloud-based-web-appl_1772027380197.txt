Develop a secure, cloud-based web application that captures employee time, manages approvals with dual electronic signatures, runs payroll compliant with Guyana 2026 labor laws, and exports payroll data to QuickBooks.

The system must include:

Admin Portal

Management Approval Portal

Employee Portal

Google Maps Geofencing Integration

Payroll Engine

Reporting Module

QuickBooks Export Functionality

3. TIME MANAGEMENT
3.1 Clock In / Clock Out

Employees must clock in and clock out.

GPS location must be captured.

Geofencing verification required.

Admin defines allowed work zone using Google Maps.

Clock-in denied if outside permitted area.

GPS coordinates stored in audit log.

3.2 Time Editing

Employees may edit time entries before submission.

All edits flagged and logged.

Full audit trail required.

4. TIMESHEET APPROVAL WORKFLOW (MANDATORY 2 SIGNATURES)

After clock-out:

Time entry appears under “My Time”.

Employee must:

Approve or Disapprove their own time.

Apply electronic signature (typed name + timestamp + IP address).

First Approver (Reporting Manager):

Approve or Reject.

Electronic signature required.

Second Approver (e.g., General Manager):

Approve or Reject.

Electronic signature required.

Rule:
No payroll processing without two management signatures.

Admin selects both approvers during employee creation.

Example:
Security Officer → Shift Supervisor → General Manager

Each timesheet must contain exactly two management signatures.

5. REQUEST MANAGEMENT

Under the “Requests” tab, employees can submit:

Leave requests

Overtime requests

Shift swap requests

Payroll disputes

Approvers and Admin can:

Approve

Reject

Comment

All actions logged.
