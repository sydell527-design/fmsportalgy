## Notes
- The application requires geolocation permissions for the Clock In/Out functionality.
- Assumes `api` and `buildUrl` from `@shared/routes` are properly exported.
- Default users from the schema will be used. Admin username standard is usually ADMIN001 or similar based on backend seed.
- Geofences are hardcoded on the client for demo purposes based on the implementation notes.
